# 🔥 Calorie Predictor App

A production-style **Machine Learning + Analytics web application** that predicts calories burned using biometric and workout data.  
This project demonstrates practical capabilities across **Data Science, Data Engineering, Analytics, and AI application development** in one end-to-end portfolio artifact.

## 🧱 Repository Components

- `app.py` — Web app serving real-time calorie predictions.
- `ML_Model_Book.ipynb` — Model development and experimentation notebook.
- `X_train.csv`, `exercise.csv`, `calories.csv` — Core data assets used in model workflow.
- `rfr.zip` — Serialized model artifact for inference.
- `render.yaml` — Deployment configuration for cloud hosting.
- `requirements.txt` — Python dependency manifest.
